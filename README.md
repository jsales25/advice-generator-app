### O que eu aprendi

Neste projeto, eu pude praticar e aprimorar minhas habilidades em:

- **Consumo de APIs:** Utilizei a função `fetch` do JavaScript para fazer requisições a uma API externa e obter dados.
- **JavaScript Assíncrono:** Aprendi a trabalhar com `async/await` para lidar com as requisições de forma mais limpa e legível.
- **Manipulação do DOM:** Utilizei JavaScript para selecionar elementos do HTML e atualizar seu conteúdo dinamicamente com os dados da API.
- **Estruturação do Projeto:** Melhorei a organização dos arquivos do projeto, separando HTML, CSS e JavaScript em suas respectivas pastas.
